## 关于我们

**EMU-Stu** 是一个由学生驱动的 GitHub 组织，专注于：

* 实验室协作
* 开源项目建设
* 技术学习与分享
* 学生创新与发展

我们为学生和实验室提供统一的空间，用于托管、管理和展示项目。

---

## 实验室

* 物联网实验室 （IOT Lab）
* 河北省安全生产与应急处置特种机器人重点实验室 （CNB Lab）
* 方舟实验室（Ark Lab）
* 利刃网安攻防实验室

> 欢迎更多实验室加入！
