# EMU-Stu

### 面向实验室、开源与技术协作的学生组织
构建项目，分享创意，共同成长。

## 关于我们
EMU-Stu 是一个由学生驱动的 GitHub 组织，专注于：
*   **实验室协同合作**
*   **开源项目开发**
*   **技术学习与分享**
*   **学生创新与实践**

我们为广大学生和实验室提供了一个统一的空间，用于托管、管理和展示技术项目。

## 下属实验室
*   **IOT 实验室**（物联网实验室）
*   **CNB 实验室**

*更多实验室即将加入。*

## 精选项目
| 项目名称 | 项目简介 |
| :--- | :--- |
| **hitscounter** | 基于 Cloudflare 的轻量级访客计数器 |
| **EMU-Stu-Portal** | 实验室与项目导航的统一门户 |
| **EMU-Stu-Brand** | 组织 Logo 及品牌视觉资产 |
